## Project title
NovaSaaS — SaaS Landing Page & Analytics Dashboard (React)

## Short tagline
A complete, responsive SaaS web app: marketing site, auth flow, and a fully
interactive analytics dashboard — built with React, Tailwind CSS & Framer Motion.

## Description (paste into Upwork "Description")
NovaSaaS is a production-quality frontend I designed and built to demonstrate
real-world UI engineering.

It includes:
• A polished landing page — animated hero, feature grid, pricing toggle, testimonials
• Authentication — login & sign-up pages with live form validation and password strength
• A complete dashboard with 6 working sections: Overview, Analytics, Revenue,
  Customers, Reports and Settings
• Interactive charts (area, bar, line, donut) and trend sparklines
• Dark / light mode that persists across the whole app
• Fully responsive (mobile → desktop) and accessible (keyboard navigation,
  ARIA labels, screen-reader support, reduced-motion)
• A custom 404 page and SPA routing

Tech: React 18, Vite, Tailwind CSS, Framer Motion, Recharts, React Router.
